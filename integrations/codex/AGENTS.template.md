# AGENTS.md Template For Codex Projects

Use installed Agency skills from `~/.codex/skills/` when the task clearly maps
to one of these specialist roles:

- `agency-backend-architect` for backend architecture, APIs, schema design, and system reliability
- `agency-frontend-developer` for UI implementation and frontend debugging
- `agency-devops-automator` for CI/CD, infrastructure, deployments, and operational automation
- `agency-reality-checker` for release readiness, validation, and evidence-based review
- `agency-security-engineer` for security review, hardening, and threat modeling

Working rules:

- Prefer the smallest set of Agency skills needed for the task.
- State which Agency skill you are using when it materially changes your approach.
- Keep outputs practical and execution-oriented.
- Do not load extra Agency skills unless the task clearly benefits from them.

Project-specific guidance:

- Add or remove Agency skill names to match this repository.
- Keep this file focused on selection rules and local constraints.
